## About the Project
This portfolio website was designed and developed from scratch to establish my professional online presence. It bridges the gap between clean **UI/UX design principles** and modern front-end engineering, focusing on responsiveness, accessibility, and smooth user interactions.

### Key Features
* **Responsive Layout:** Fully optimized for seamless viewing across mobile, tablet, and desktop screens.
* **Component-Driven Structure:** Written with organized, scalable, and easy-to-maintain code.
